### 🎒 "Plan your journey, one click at a time!" 🎒

Welcome to the **Trip Planner**! This is a full-stack web application designed to help travelers organize their itineraries easily.

---

## 🚀 Key Features 🚀

* **🔐 Secure Login:** Authentication system with **OTP Email Verification** (via Nodemailer).
* **📍 Trip Management:** Create, view, update, and delete your travel plans effortlessly.
* **📱 Responsive Design:** Fully optimized for both Desktop and Mobile views.
* **⚡ Real-time Updates:** Fast and snappy UI powered by React.

---

## 🛠️ Tech Stack (The Engines) 🛠️

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | ⚛️ React.js | Dynamic & Interactive UI |
| **Backend** | 🟢 Node.js / Express | Robust API Logic |
| **Database** | 🍃 MongoDB | Flexible Data Storage |
| **Auth** | 🔑 JWT & Nodemailer | Secure Sessions & OTPs |

## 🏗️ How to Set Up 🏗️
### 1️⃣ Clone the repo
```bash
git clone [https://github.com/pradipkumar271/Trip-Planner-MERN-App.git](https://github.com/pradipkumar271/Trip-Planner-MERN-App.git)
```
### 2️⃣ Install Dependencies
**For Backend:**
```bash
cd backend
npm install
```
**For Frontend:**
```bash
cd ../frontend
npm install
```

### 3️⃣ Environment Setup 🔑
Create a `.env` file in the **backend** folder and add:
* `MONGO_URI` = (Your MongoDB link)
* `MAIL_USER` = (Your email for OTP)
* `MAIL_PASS` = (Your app password)

### 4️⃣ Run the App 🚀
```bash
# In backend folder
npm run dev
# (Note: Use npm start if you don't have a dev script configured)

# In frontend folder
npm start
```
